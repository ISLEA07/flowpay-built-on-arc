# FlowPay

Programmable Payments Built on Arc

## Overview

FlowPay is a payment infrastructure application built on Arc Testnet.

The platform enables users to:

- Connect Wallet
- Send Payments
- Escrow Transactions
- Revenue Splitting
- Track On-chain Activity
